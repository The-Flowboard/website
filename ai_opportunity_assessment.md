# AI OPPORTUNITY ASSESSMENT TOOL
## Lead Generation Self-Assessment for AI Consultancy Website

---

## 🎯 TOOL OVERVIEW

**Purpose:** Lead generation tool that identifies 3-5 high-impact AI opportunities for business owners

**User Experience Flow:**
1. User answers 15-20 strategic questions (5-7 minutes)
2. Algorithm scores answers and matches to opportunity database
3. User receives personalized PDF report with:
   - 3-5 ranked AI opportunities
   - Cost estimates and projected ROI
   - Preliminary implementation roadmap
   - CTA to book full discovery call

**Technical Approach:** Algorithm-based (no LLM required) using weighted scoring and opportunity templates

## 🧮 SCORING ALGORITHM

### STEP 1: CALCULATE OPPORTUNITY SCORES

For each of the 20+ pre-defined opportunities in the database, calculate:

```
Opportunity Score = (Pain Score × Pain Weight) + (Strategic Fit × Strategic Weight) + (Readiness Bonus) - (Complexity Penalty)
```

#### PAIN SCORE CALCULATION

Based on Q5-Q10 responses:

```
Pain Score = User's Rating (0-5) for the relevant question

Example:
- Customer Support AI opportunity: Uses Q5 score
- Content Generation opportunity: Uses Q6 score
- Lead Qualification opportunity: Uses Q7 score
- Document Processing opportunity: Uses Q8 score
- Knowledge Base opportunity: Uses Q9 score
- Workflow Automation opportunity: Uses Q10 score
```

**Pain Weight:** 40% (highest weight - actual pain drives ROI)

---

#### STRATEGIC FIT CALCULATION

Based on Q11-Q12 responses:

```
Strategic Fit Score:
- If opportunity aligns with Primary Goal (Q11): +3 points
- If opportunity aligns with Value Priority (Q12): +2 points

Strategic Alignment Examples:
- Primary Goal = "Reduce costs" → Document Processing gets +3
- Primary Goal = "Improve customer satisfaction" → Customer Support AI gets +3
- Value Priority = "Freeing up employee time" → Content Generation gets +2
- Value Priority = "Faster customer response" → Chatbot gets +2
```

**Strategic Weight:** 30%

---

#### READINESS BONUS

Based on Q4, Q15, Q16:

```
Readiness Score:
- Tech Comfort (Q4):
  - Very comfortable: +2
  - Moderately comfortable: +1
  - Somewhat comfortable: 0
  - Not very comfortable: -1

- Executive Sponsor (Q15):
  - C-level committed: +3
  - VP/Director committed: +2
  - Need to build case: +1
  - No sponsor: -2 (RED FLAG)

- Data Organization (Q16):
  - Well-organized: +2
  - Somewhat organized: +1
  - Not organized: 0
  - Paper-based: -2 (RED FLAG)

Total Readiness Bonus: Sum of above (max +7)
```

**Readiness Weight:** 20%

---

#### COMPLEXITY PENALTY

Based on Q13 (budget) and Q14 (timeline):

```
Complexity Penalty:
- If opportunity cost > user's budget: -5 points (eliminate from consideration)
- If opportunity timeline > user's timeline expectation: -2 points

Budget Tiers (from opportunity database):
- Tier 1: $5K-$15K
- Tier 2: $15K-$30K
- Tier 3: $30K-$60K
- Tier 4: $60K-$100K
- Tier 5: $100K+

Timeline Tiers:
- Quick Win: 4-8 weeks
- Standard: 8-12 weeks
- Moderate: 12-20 weeks
- Complex: 20+ weeks
```

**Complexity Weight:** 10%

---

### STEP 2: RANK OPPORTUNITIES

```
1. Calculate final score for each opportunity:
   Final Score = (Pain Score × 0.4) + (Strategic Fit × 0.3) + (Readiness × 0.2) - (Complexity × 0.1)

2. Sort opportunities by Final Score (descending)

3. Select Top 5 opportunities with score > threshold (e.g., 3.0/5.0)

4. If fewer than 3 opportunities score > 3.0:
   - Lower threshold to 2.0
   - Include note in report: "Your readiness assessment suggests starting with foundational improvements"
```

---

### STEP 3: PERSONALIZE COST & ROI ESTIMATES

For each selected opportunity, adjust cost and ROI based on company size (Q1) and revenue (Q3):

```
Cost Multiplier by Company Size:
- Micro (1-10): 0.7× base cost
- Small (11-50): 1.0× base cost (baseline)
- Medium (51-200): 1.3× base cost
- Large (201-1000): 1.8× base cost
- Enterprise (1000+): 2.5× base cost

ROI Multiplier by Company Size:
- Micro: 0.6× base ROI (smaller volumes)
- Small: 1.0× base ROI (baseline)
- Medium: 1.5× base ROI
- Large: 2.5× base ROI
- Enterprise: 4.0× base ROI
```
